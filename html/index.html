HTML - Hypertext Markup Language


HTML is a language made of elements which can be applied to pieces of text to give them different meaning in a document, structure page into logical sections and embed content such as images and videos into a page.

  • Hypertext - link to page on same or differenet website.
  • Tags are not case sensitive - lower case is convention.
  • The HTML parser reduces each sequence of whitespace to a single space when rendering the code.
  • An element is written as below;
    •  <p> An element made up of opening and closing tag </p>

    Nesting of elements

  • An element can be placed inside another element.
    •  <p> Opening and closing <strong> tags  </strong> inside the parent element. </p>

    Block vs Inline elements

    Tags are of two types block and inline.

    Block Level

  • Block level elements occupy entire width.
  • Start on new line and push the preceeding element to new line.

    • Inline Elements

  • Take up the width required for the content.
  • Inline elements can start anywhere in a line.

    • Void Elements

  • Some elements consist of a single tag, which are used to insert/embed something in the document.

    • Atrributes

    Attributes contain extra information about the element that won't appear in the content.

    Atributes can be global or element specific

    An atrribute should have:

  • A space between it and the element name. (For an element with more than one attribute, the attributes should be separated by spaces too.)
  • The attribute name, followed by an equal sign.
  • An attribute value, wrapped with opening and closing quote marks.

    • Sometimes you will see attributes written without values they are called boolean attributes.

    Atrributes can be written using a single/double quotes.

    Atrributes can be written without quotes if only single attribute is to be written.

    Anatomy of HTML document

            <!DOCTYPE html>
    
        <html lang="en-US">
    
        <head>
    
            <meta charset="utf-8" />
    
            <title>My Webpage</title>
    
        </head>
    
        <body>
    
            <p>This is my page</p>
    
        </body>
    
        </html>
