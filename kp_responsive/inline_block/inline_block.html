Inline-Block and the Button styling

Display property as inline-block gives us the best of both block and inline world properties.

Inline-block use case for link turned to button. Always use padding to create the size of the button and not the width and height. Always put the class on the link directly.

Left and right padding needs to be bigger than top and bottom use 1:2 ratio.
