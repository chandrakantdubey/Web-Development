Working with the padding and the margins.

  • All block elements have margin which is equal to the font size, except for the heading elements.
  • To center the element in the parent use left right margin auto.
  • Two margin touching colapse to a single margin, with the bigger one getting applied.
  • The parent element margin and first child element margin get colapsed to each other, hence it is better to use padding and set the padding top and bottom to zero to avoid any issues.

    • Working with the inline text and tags.

  • Only inline elements can be nested inside another inline element.
  • Nesting a strong tag which is inline element insided a block element.
  • Inline element respect margin and padding and border for right and left side only and not for top and bottom.
  • Nesting a inline inside another inline element anchor tag.
  • A span tag is used for styling text and has no default styling and no semantic meaning.

    • Play with the link and the states of the link.

    A link has different states like:
    Link: default state
    Visited: clicked link
    Hover: on mouse over
    Focus: on tab
    Active: on click

  • The states of the link are to be applied in the same order for them to work.
  • For good website hover & focus should always be there along with the link state.
